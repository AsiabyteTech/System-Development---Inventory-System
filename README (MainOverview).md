# Project Name: [Asiabyte P&L Inventory System]

## Overview
This project is an inventory system created to keep track on the hands on inventory of Asiabyte Technologies.  
It includes front-end, back-end, database, and integration components to provide complete system functionality.

## Features
- Real-time monitoring via web 
- Front-End: User-friendly interface
- Back-End: API-driven server logic
- Database: Secure storage and retrieval
- Notification system for events (e.g., alerts)
